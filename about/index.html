关于Milo研究中心

  Milo研究中心自豪地采用Hexo + Github Pages建站。

  Milo折腾过戈戈主机，Bluehost，庙，DigitalOcean，腾讯云等虚拟主机或者云服务器。学费交的差不多的时候，Milo发现原来最重要也最难的是坚持创作。于是，Milo决定使用Github做独立博客，主题选最多人使用的next主题。这么做的目的一方面是为了专注创作，另一方面是希望未来能够看到Milo的Github页面是绿油油一片。

  本站内容主要涉及以下3个方面：日常代码研究，人工智能，个人项目经验等等。其中，日常代码研究主要记录工作时碰到的难题或者业余时间研读过的源码；人工智能方向是未来N年的重点，Milo认为应该设置单独的板块；个人项目经验则会记录Milo在业余时间完成的项目。
